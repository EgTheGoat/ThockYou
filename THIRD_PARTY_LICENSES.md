# Third-Party Licenses

ThockYou bundles audio sample packs and configuration files derived from the
[Mechvibes](https://github.com/hainguyents13/mechvibes) project. The original
files are distributed under the MIT License, reproduced below.

The following sound packs in `Resources/SoundPacks/` are derived from
Mechvibes' built-in packs (audio files have been transcoded from `.ogg` to
`.wav` for use with AVAudioEngine; configuration JSON files are unchanged):

- `CherryBlack/`   — `cherrymx-black-pbt`
- `CherryBlue/`    — `cherrymx-blue-pbt`
- `CherryBrown/`   — `cherrymx-brown-pbt`
- `CherryRed/`     — `cherrymx-red-pbt`
- `EgOreo/`        — `eg-oreo`
- `EgCrystalPurple/` — `eg-crystal-purple`
